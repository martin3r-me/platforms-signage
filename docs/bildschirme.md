# 🖥️ Bildschirme

Ein Bildschirm ist ein gekoppeltes Anzeigegerät — typischerweise ein **Fire TV Stick**, aber jeder Browser im Vollbild funktioniert.

---

## Fire TV einrichten (in Kürze)

1. **App laden** — über den Download-Button bzw. die **Downloader-App** auf dem Fire TV.
2. **Installation erlauben** — „Apps unbekannter Herkunft" für Downloader aktivieren.
3. **Installieren & öffnen** — die App zeigt einen großen **Kopplungs-Code**.
4. **Koppeln** — den Code im Modul unter **Bildschirme → Bildschirm koppeln** eingeben.

Die richtige Server-Adresse steckt bereits in der heruntergeladenen App — kein Eintippen nötig. Eine ausführliche Schritt-für-Schritt-Anleitung (inkl. Entwickleroptionen) findest du direkt auf der Bildschirme-Seite.

### Download per Code freischalten

Damit die Downloader-App die Installationsdatei **ohne Login** laden kann, erzeugst du im Dashboard einen **4-stelligen Code**. Daraus entsteht eine kurze Adresse (z. B. `…/firetv/1234.apk`), die du im Downloader eingibst. Der Code ist zeitlich begrenzt gültig; eingeloggte Admins laden die App auch direkt ohne Code.

---

## Einstellungen eines Bildschirms

- **Name** und **Ausrichtung** (Quer-/Hochformat, auch 180° gedreht).
- **Standard-Wiedergabeliste** — läuft, wenn kein Zeitplan greift.
- **Hintergrundmusik** — eine Musik-Liste oder ein einzelner Stream/Audiotitel.
- **Zeitpläne** — beliebig viele kombinierbar (siehe [Zeitpläne](zeitplaene)); sie übersteuern die Standard-Liste in ihren Zeitfenstern.
- **Zeitzone** — Grundlage für die zeitgesteuerte Umschaltung.
- **„Was läuft gerade?"** — zeigt den aktuell aktiven Plan + Liste.

> Änderungen wirken automatisch: Der Player erkennt neue Inhalte beim nächsten Abruf und lädt neu. Mit **Neuladen** kannst du das sofort auslösen.

---

## Live-Vorschau

Auf der Detailseite siehst du eine **Live-Vorschau** des Bildschirms und kannst ihn per **Vollbild öffnen** in einem neuen Tab betrachten.

---

## Eltern-/Kind-Teams

Bist du in einem **übergeordneten Team**, zeigt dir das **Dashboard** zusätzlich die Bildschirme der **Kind-Teams** — mit Team-Kennzeichnung und Online-Status, rein zum **Überblick** (z. B. „auf einen Blick: sind alle online?"). **Bearbeitet** wird jeder Bildschirm nur in seinem eigenen Team.
